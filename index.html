MIPS Assembly Random Number Generator

This project introduced me to lower level languages and their application to
physical electronics. MIPS uses registers which correlate to memory directly
on the chip. I used an *algorithm which simulated a random generator.

The algorithm involved used a seed which made it look like the numbers were random however,
when using the same seed the same numbers would be the output. In MIPS it is impossible to get a truely random generator.
